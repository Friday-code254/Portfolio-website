# My Web Project

This project is a simple web application that includes various pages for user interaction, including Home, Login, Registration, Personal, and Exit pages.

## Project Structure

```
my-web-project
├── src
│   ├── css
│   │   ├── styles.css        # Contains styles for the web project
│   ├── js
│   │   ├── main.js           # Contains JavaScript for user interactions
│   ├── pages
│   │   ├── home.html         # Home page with main content
│   │   ├── login.html        # Login page for user authentication
│   │   ├── registration.html  # Registration page for new users
│   │   ├── personal.html      # Personal page for user-specific information
│   │   └── exit.html         # Exit page for logout confirmation
├── index.html                # Entry point of the web project
└── README.md                 # Documentation for the project
```

## Features

- User authentication through Login and Registration pages.
- Dynamic content updates using JavaScript.
- User-specific information displayed on the Personal page.
- Clean and responsive design using CSS.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the application.
3. Ensure that you have a local server running if you want to test JavaScript functionalities.

## Usage Guidelines

- Navigate to the Home page to access links to other pages.
- Use the Login page to enter your credentials.
- Register a new account on the Registration page.
- After logging in, view your information on the Personal page.
- Use the Exit page to log out of the application.
